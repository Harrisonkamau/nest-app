## Description

A todos app created in Nest.js and MongoDB.

## Installation

```bash
$ npm install
```

Start a Mongod server by opening a terminal run: `mongod`. Ensure you have MongoDB driver installed on your PC. Else, follow these instructions [here](https://docs.mongodb.com/manual/installation/).

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Routes
The app runs on port: `3000`

`home`: `/`
All todos: `/todos`

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
